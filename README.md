北京二手房分析可视化平台
该平台主要分为三个模块
数据分析模块、后端开发模块、数据可视化模块
使用Flume+HDFS+Spark SQL+Spark MLlib+Datax+MySQL框架和HTML+JS(Echart)+CSS进行数据可视化大屏的开发。
项目运行步骤：
有完整Hadoop框架的虚拟机
安装并配置Tomcat
在IDEA中创建相应项目
为Web部分安装Echarts包
根据pom.xml配置文件加载配置
